Tic Tac Toe

Rules of the Game

Basic Rules:

  1. There are 2 players: Player 1 uses X and Player 2 uses O.
  2. The game is played on a 3 × 3 grid.
  3. Players take turns placing their symbol in an empty square (Player 1 goes first).
  4. Once placed, a symbol cannot be changed.
  5. The first player to get 3 in a row (horizontally, vertically, or diagonally) wins.
  6. If all 9 squares are filled and no one wins, the game ends in a draw.
  7. After each game, clear the entire board before starting again.
